核心功能

商品销量实时监控

按关键词或商品ID抓取小红书商品详情页的销量数据，支持历史销量趋势分析和未来30天销量预测。

爆款笔记分析

采集爆款笔记的互动数据（点赞/收藏/评论）、流量来源（搜索/推荐占比）及标签话题热度。

达人数据评估

批量导出达人主页公开数据，分析达人历史合作品牌的CPE、CPM等商业化数据，评估带货效果。

带货转化分析

分析达人直播带货的UV价值、订单转化漏斗（曝光-点击-下单），量化带货效果。

行业榜单

按行业/品类筛选商品销量TOP100榜单，支持按时间维度回溯历史数据。

定时任务

可配置定时任务自动采集数据，支持云采集服务批量导出数据，提高工作效率。

API文档

GET /api/v1/product/sales?product_id={product_id}&start_date={start_date}&end_date={end_date}
获取指定商品的历史销量数据，支持时间范围筛选。

请求参数

参数 类型 是否必填 描述
product_id string 是 小红书商品ID
start_date string 否 起始日期，格式YYYY-MM-DD
end_date string 否 结束日期，格式YYYY-MM-DD

响应示例

  "data": {
    "product_id": "5eb1097ba091410953951d17",

      // ...更多数据

    }
  }
}
POST /api/v1/kols/search
根据条件搜索小红书达人，支持多种筛选条件。

请求参数

参数 类型 是否必填 描述
fans_min integer 否 粉丝数下限
fans_max integer 否 粉丝数上限
category string 否 内容分类，多个用逗号分隔
cpe_max float 否 最大互动成本
page integer 否 页码，默认为1
page_size integer 否 每页数量，默认为20
